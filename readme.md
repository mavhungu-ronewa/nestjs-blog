## Blog Application
Application to share knowledge and insight
## Requirements
```bash
1. Nodejs
2. Mongodb Atlas Account
```
## Getting started
```bash
1. git clone https://github.com/mavhungu/nestjs-blog
2. cd nestjs-blog
3. Run npm run install / yarn install
4. Run npm run start-dev
```
## Stack used
```bash
Backend - Nestjs
Frontend - React
```
